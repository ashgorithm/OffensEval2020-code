========================

OffensEval 2020: Multilingual Offensive Language Identification in Social Media (SemEval 2020 - Task 12)
Test data - Sub-task B
v 1.0: February 26 2020
https://sites.google.com/site/offensevalsharedtask/home

========================

1) DESCRIPTION

The file test_b_tweets.tsv contains 1422 unlabeled tweets. This file contains ONLY tweets which are offensive.

You are required to upload your sub-task B predictions for each of the 1422 instances to CodaLab by no later than 4 Mar 2020 (23:59 GMT).

You will find ALL the necessary information regarding data format, dates, number of submissions, etc. at CodaLab. Please read it carefully.

2) FORMAT

Instances are included in TSV format as follows:

id	tweet

3) TASK AND LABELS

(B) Sub-task B: Automatic categorization of offense types

- (TIN) Targeted Insult and Threats - A post containing an insult or threat to an individual, a group, or others (see categories in sub-task C).
- (UNT) Untargeted - A post containing non-targeted profanity and swearing.

Posts containing general profanity are not targeted but they contain non-acceptable language.

4) SUBMISSION

When you have your model predictions, you should upload a CSV file on CodaLab in the following format. 

ID, LABEL
